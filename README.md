### 🏃‍♀️‍➡️ Run locally

Woud like to run this project locally? Open terminal and follow these steps:

1. Clone the repo
    ```sh
    git clone https://aidas415.github.io/sibar/
    ```
2. Install NPM packages
    ```sh
    npm i
    ```
    or
    ```sh
    npm install
    ```
2. Run the server
```sh
npm run dev
```
